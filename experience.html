MY EXPERIENCE WITH FULL STACK DEVELOPMENT

Terminal Programs

Terminals are text-based prompt environments with an input and output. Its important for Developers as it allows us to enter commands or prompts that the computer can process, perform and display the result. Although its the most bare-bones way to interact with a computer, its simplicity is also its strength.

My first exposure to Terminal programs was at Coder Academy and I now have a good understanding of basic command line concepts and common terminal commands. When creating this website I used Ubuntu 22.10 for my Terminal.

HTML 5 and CSS 3

Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS) are two of the cornerstone technologies that make up all web pages. HTML is often viewed as the skeleton struction of a web page and often holds static or raw content. That's were CSS comes in, and by applying style sheets you are able to create aesthetically pleasing web pages through visual and layout styling.

Through my understanding and application of HTML and CSS, I was able to build this very Portfolio. Underneath it all, you will find that I have used Semantic HTML, whilst also having an understanding of different elements and structures. To style the website in CSS I had to use various syntax and selectors and needed a great understanding of Flexbox modelling, Display positioning and Responsive Design. I can safely openly admit that there were some struggles with the build, but because of that, I am confident in troubleshooting and optimising code.

Git

Git is a Version control system that allows us to track changes and back-up files in a project. Understanding and using Git is essential for any IT Developers, as it greatly facilitates group work. It can help increase work-flow speed, can be securely stored online and supports non-linear workflows.

I not only have a basic understand of git concepts, but I am able to implemenet version control with git commands in Terminal Programs. Whilst building my portfolio, I applied version control techniques to keep track of changes, work on a local and remote repo, and updated my project from my desktop and laptop when required.
